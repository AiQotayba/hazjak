# React Hazjak - حجز ملاعب كرة القدم في حلب

مشروع React حديث لبناء منصة حجز ملاعب كرة قدم في حلب باستخدام أحدث التقنيات.

## الميزات

- ⚡ **React 18** مع TypeScript لتجربة تطوير ممتازة
- 🎨 **Tailwind CSS** للتصميم السريع والمرن
- 🔄 **Context API** لإدارة الحالة العامة
- 📅 **date-fns** للتعامل مع التواريخ
- 🎯 **Lucide React** للأيقونات الحديثة
- 📱 **تصميم متجاوب** يعمل على جميع الأجهزة
- 🌙 **دعم اللغة العربية** مع اتجاه RTL
- 🔐 **نظام مصادقة** محلي مع localStorage
- 📅 **نظام حجز متقدم** مع تقويم تفاعلي وفترات زمنية
- 🎭 **تجربة مستخدم سلسة** مع animations وtransitions

## البنية التقنية

```
src/
├── components/          # مكونات React
│   ├── Header.tsx      # شريط التنقل العلوي
│   ├── Hero.tsx        # القسم الرئيسي
│   ├── Fields.tsx      # عرض الملاعب
│   ├── BookingModal.tsx # modal الحجز
│   ├── AuthModal.tsx   # modal المصادقة
│   └── Footer.tsx      # التذييل
├── contexts/           # Context API لإدارة الحالة
│   ├── AuthContext.tsx # إدارة المصادقة
│   └── BookingContext.tsx # إدارة الحجوزات
├── App.tsx            # المكون الرئيسي
├── main.tsx           # نقطة الدخول
└── index.css          # أنماط Tailwind
```

## التثبيت والتشغيل

### متطلبات النظام
- Node.js (إصدار 16 أو أحدث)
- npm أو yarn

### خطوات التثبيت

1. **تثبيت التبعيات:**
   ```bash
   npm install
   ```

2. **تشغيل المشروع في وضع التطوير:**
   ```bash
   npm run dev
   ```

3. **بناء المشروع للإنتاج:**
   ```bash
   npm run build
   ```

4. **معاينة البناء:**
   ```bash
   npm run preview
   ```

## الميزات الرئيسية

### 🏟️ عرض الملاعب
- عرض جميع الملاعب مع الصور والتقييمات
- معلومات مفصلة عن كل ملعب (الموقع، السعر، التقييم)
- fallback للصور في حالة عدم تحميلها

### 📅 نظام الحجز المتقدم
- تقويم شهري تفاعلي
- عرض الحجوزات الموجودة
- اختيار الوقت من فترات محددة (08:00 - 21:30)
- عرض حالة كل وقت (متاح، محجوز، عربون)

### 🔐 نظام المصادقة
- تسجيل الدخول وإنشاء الحساب
- حفظ البيانات محلياً باستخدام localStorage
- حماية المسارات التي تحتاج مصادقة

### 📱 تصميم متجاوب
- يعمل على الهواتف، التابلت، والحواسيب
- تصميم RTL للغة العربية
- animations سلسة وtransitions

## التقنيات المستخدمة

- **React 18** - مكتبة JavaScript للواجهات
- **TypeScript** - JavaScript مع أنواع البيانات
- **Vite** - أداة بناء سريعة
- **Tailwind CSS** - إطار عمل CSS
- **Lucide React** - مكتبة أيقونات
- **date-fns** - مكتبة التواريخ
- **Context API** - إدارة الحالة في React
