# 🚀 دليل الاختبار السريع

## اختبار التسجيل:

```bash
# 1. شغّل الـ Server
cd backend
node server.js

# 2. شغّل اختبار التسجيل في terminal آخر
node test-register.js
```

## اختبار OTP:

```bash
# شغّل اختبار OTP
node otp-test.js
```

## اختبار من المتصفح:

```bash
# افتح الموقع
http://localhost:5000

# أو صفحة اختبار OTP
http://localhost:5000/backend/test-otp.html
```

## الملفات المُنشأة:

- ✅ `test-register.js` - اختبار التسجيل
- ✅ `otp-test.js` - اختبار OTP
- ✅ `test-otp.html` - صفحة اختبار تفاعلية
- ✅ `REGISTER_FIX.md` - توثيق إصلاح التسجيل
- ✅ `OTP_GUIDE.md` - دليل OTP الشامل
